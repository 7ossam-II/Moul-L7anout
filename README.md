# Moul7anout — Landing Page (Pro Max)

## 🐳 Zero-Config Preview (Docker)
This branch contains the high-end "Clean Digital Souk" marketing site. To see the "perfect" version with all animations and fonts identical to the master design, use Docker.

### How to Run:
1. **Pull the latest code**:
   ```bash
   git pull origin website
   ```
2. **Launch with Docker**:
   ```bash
   cd website
   docker-compose up --build
   ```
3. **View**: Open [http://localhost:3000](http://localhost:3000)

## 🎨 Design Specs
- **Colors**: Teal Blue (#0F4C81), Vibrant Orange (#FF6B35).
- **Typography**: Syne (Headlines), Plus Jakarta Sans (Body).
- **Tech**: Next.js 14, Framer Motion, Tailwind CSS.
