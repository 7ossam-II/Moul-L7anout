# Moul7anout — Developer Hub

## 🛠 Getting Started
Welcome to the core development branch. This is where the "Moul7anout" app (Buyer, Seller, Backend) is built.

### 1. Clone & Setup
```bash
git clone https://github.com/7ossam-II/Moul-L7anout.git
cd Moul-L7anout
git checkout develop
```

### 2. Project Structure
- **/backend**: Node.js/Express API with Prisma/Drizzle.
- **/frontend**: Next.js 14 App Router (Buyer & Seller Dashboard).
- **/docs**: API Contracts and Database Schemas.

### 3. Workflow (10x Standards)
- **Branching**: Create feature branches from `develop` (e.g., `feature/taha-seller-logic`).
- **Design**: All UI must follow the **UI UX PRO MAX** standards defined in the team guide.
- **i18n**: No hardcoded strings. Use the `messages/` JSON files.
